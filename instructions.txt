Go to github. Select New

After creating repository, go to cmd, 

git clone 'repository link' //repository ka local copy banata

git add .// Git check kar raha hai ki kaunse files hai

git commit -m "This is my first commit" //Means git ne commit kiya hai.. Woh ab ready hai push krne

git push //This pushes the files inside the repository

git pull //to pull any changes made on the github
